## Categories

### API Errors

Common API error codes and their solutions.

### Response Issues

Problems with model responses and output.

### Tool/Function Issues

Problems with tool use and function calling.

### Performance Issues

Latency, timeouts, and optimization.

## Adding a Troubleshooting Guide

Use the [how-to template](../_templates/how-to-template.md) with focus on:

1. **Error** - The exact error message or symptom
2. **Cause** - Why this happens
3. **Solution** - How to fix it
4. **Prevention** - How to avoid it

See [CONTRIBUTING.md](../CONTRIBUTING.md) for guidelines.
